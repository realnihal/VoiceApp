API Documentation With RSA Encryption

IITM ASR (STT) API

  • This documentation contains information regarding the TTS API and its usage.This API is build and developed by Speech Lab IITM.
  • It supports 7 languages:

english, gujarati, hindi, kannada, marathi, tamil, and telugu

  • Interested students can check out this link https://asr.iitm.ac.in/asr/v2

API Request Example:

  • Endpoint : https://asr.iitm.ac.in/asr/v2/decode
  • Use postman (https://www.postman.com/) to get template codes for programming language of your choice
  • Python example given below:

Python

# STT IITM SPEECH LAB
    import requests
    import json
    
    input_language = 'hindi' # input audio language
    input_audio_path = './input.mp3' # input audio
        
    url = "https://asr.iitm.ac.in/asr/v2/decode" # endpoint
    audio_file = input_audio_path.split('/')[::-1][0] # extracting the audio file name
    payload={'vtt': 'true', 'language': input_language} # json payload
    files=[('file',(audio_file,open(input_audio_path,'rb'),'application/octet-stream'))] # audio file in binary format
    headers = {}
    response = requests.request("POST", url, headers=headers, data=payload, files=files)
    text = json.loads(response.text)
    stt = text["transcript"] # transcript only
    print(response.text) # complete response
    print(stt) # transcript only

IITM TTS API

  • The documentation contains information regarding the TTS API and its usage. This API is build and developed by Speech Lab IITM.
  • It supports 15 languages

Assamese, Bengali, Bodo, Gujarati, Hindi, Kannada, Malayalam, Manipuri, Marathi, Odia, Punjabi, Rajasthani, Tamil, Telugu, and Urdu

API Response Example

  • Endpoint : https://asr.iitm.ac.in/ttsv2/tts
  • Use postman (https://www.postman.com/) to get template codes for programming language of your choice.

Python Example

# TTS IITM SPEECH LAB
    import requests
    import json
    import base64
      
    gender = 'male'
    lang = 'Hindi'
    text = 'चॉकलेट आइसक्रीम बनाने की विधि बताओ'
    
    url = "https://asr.iitm.ac.in/ttsv2/tts" # Endpoint
    # message payload
    payload = json.dumps({
    "input": text,
    "gender": gender,
    "lang": lang,
    "alpha": 1,
    "segmentwise":"True"
    })
    
    headers = {'Content-Type': 'application/json'}
    response = requests.request("POST", url, headers=headers, data=payload).json()
    
    audio = response['audio'] # encoded speech data
    file_name = "tts.wav" # output file name
    wav_file = open(file_name,'wb') # creating an empty file
    decode_string = base64.b64decode(audio) # decoding audio file
    wav_file.write(decode_string) # writing decoded audio data to a wave file "tts.mp3"
    wav_file.close()
  • The audio file will be saved in the current working directory as “tts.wav”

RP_Bank API

The following message formats are accepted at server for decryption. Each team will be provided a token id which should be used . The server accepts encrypted text which is decoded and then sent the response in encrypted format.Participants will be provided with a private and a public Key . The Public Key will be used to encrypt the data in server and the user side for the data that needs to be transmitted. Private Key will be used to decrypt the data in the server and the user end for the data that was received.

The Encryption used is RSA Encryption with cipher = PKCS1_v1_5 .The message data sent from the server is encrypted and then encoded in base64 format. Format of the encryption will be given below

encryptedData = Base64Encode(RSA/PKCS1Encryption(data,PubKey))   ------------ Encryption


decrypted_Data = Base64Decode(RSA/PKCS1Decryption(encryptedData,PrivateKey))   ------------ Decryption

import requests
import json
import base64 import b64encode
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_v1_5
    
url = 'https://events.respark.iitm.ac.in:3000/rp_bank_api'
    
# to check balance
payload = json.dumps({
		"action":"balance",
    "token":"oeo9cjc5n9lknivgu0au",
    "nick_name":"cindy"
})


def rsa_encrypt(payload_data):
    key = open("public.pem").read()
    key = key.replace("-----BEGIN PUBLIC KEY-----", "").replace("-----END PUBLIC KEY-----", "").replace("\n", "")
    key = b64decode(key)
    key = RSA.importKey(key)

    cipher = PKCS1_v1_5.new(key)
    ciphertext = b64encode(cipher.encrypt(bytes(payload_data, "utf-8")))

    return ciphertext
    
headers = {
        'Content-Type': 'application/text'
    }

data_encrypted = rsa_encrypt(payload)
    
response = requests.request("POST", url, headers=headers, data=data_encrypted)
print(response.text) # The data response will be encrpyted
